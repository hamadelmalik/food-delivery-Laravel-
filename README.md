# 🍔 Food Delivery System (Laravel)

A Food Delivery web application built with Laravel and MySQL.

## 🚀 Features

### 👤 User Side
- User authentication (Register / Login)
- Browse food items
- Add to cart
- Place orders
- View order history

### 🧑‍💼 Admin Panel
- Manage products (Add / Edit / Delete)
- Manage categories
- Manage orders
- Manage users
- Dashboard management

---

## 🛠️ Tech Stack
- Laravel
- MySQL
- Blade Templates
- Bootstrap / Tailwind

---
